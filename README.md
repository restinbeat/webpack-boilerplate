# webpack-boilerplate
practice webpack

### Check node version

```bash
node -v
npm -v
```

## 1. NPM Commands

### How to create package.json

```bash
npm init
or
npm init -y
```

### Install library

```bash
npm i
npm install <package_name> (--save-prod)
or 
npm install <package_name> --global, -g > <package_name>

npm uninstall <package_name>

```
